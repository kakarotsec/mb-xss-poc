Impact

  • One-click XSS on the main business portal (40,000+ businesses)
  • Full session hijacking — steal OIDC tokens, cookies, API keys
  • Persistent — localStorage poisoning survives page refreshes
  • Chainable with Branch.io open redirect for higher credibility
  • No authentication required — works against unauthenticated AND authenticated users

Root Cause

1. import-map-overrides@2.2.0 loaded in PRODUCTION (dev-only library)
2. No Content-Security-Policy header
3. SystemJS module loader trusts localStorage overrides
4. Cloudflare WAF bypassed via URL-encoding the ?imo= JSON parameter

Remediation

1. CRITICAL: Remove import-map-overrides.js from production
2. CRITICAL: Add Content-Security-Policy: script-src 'self' https://cdn.jsdelivr.net
3. HIGH: Add WAF rule for ?imo= parameter (even URL-encoded variants)
